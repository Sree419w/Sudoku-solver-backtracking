# Sudoku Solver Using Backtracking

This project solves a 9x9 Sudoku puzzle using the backtracking algorithm.

## Features
- Uses recursion and backtracking
- Validates row, column, and 3x3 grid constraints
- Implemented in Python

## How to Run
1. Install Python 3.x
2. Run the file:
   python sudoku_solver.py
